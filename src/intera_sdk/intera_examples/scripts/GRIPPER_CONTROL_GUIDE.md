# راهنمای کنترل گیره و مفاصل ربات Sawyer

## اجرای برنامه
```bash
cd /home/ali/Downloads/ubuntu/sawyerws_v1
source devel/setup.bash
rosrun intera_examples joint_control_interface.py
```

## کنترل مفاصل (Joint Control)

### کنترل با کیبورد:
- **1/q**: j0 (چرخش پایه) - افزایش/کاهش
- **2/w**: j1 (بالا/پایین شانه) - افزایش/کاهش  
- **3/e**: j2 (آرنج) - افزایش/کاهش
- **4/r**: j3 (چرخش ساعد) - افزایش/کاهش
- **5/t**: j4 (خم شدن مچ) - افزایش/کاهش
- **6/y**: j5 (چرخش مچ) - افزایش/کاهش
- **7/u**: j6 (چرخش انتهایی) - افزایش/کاهش

### کنترل با ماوس:
1. روی دکمه "Select" کنار هر مفصل کلیک کنید
2. با چرخاندن چرخ ماوس آن مفصل را کنترل کنید

## کنترل گیره (Gripper Control)

### کلیدهای کیبورد:
- **o**: باز کردن گیره (Open) - برای رها کردن اشیاء
- **p**: بسته کردن گیره (Close) - برای گرفتن اشیاء  
- **c**: کالیبره کردن گیره (Calibrate)

### دکمه‌های رابط گراfikی:
- **Open Gripper (O)**: باز کردن گیره
- **Close Gripper (P)**: بسته کردن گیره
- **Calibrate (C)**: کالیبره کردن گیره

## راهنمای برداشتن باکس (Pick and Place)

### برداشتن اشیاء:
1. **موقعیت‌یابی**: ربات را بالای شیء حرکت دهید
2. **پایین آوردن**: ربات را به سطح شیء پایین بیاورید
3. **گرفتن**: دکمه **P** را بزنید تا گیره بسته شود
4. **بلند کردن**: ربات را با شیء بلند کنید

### رها کردن اشیاء:
1. **حرکت**: ربات را به مکان مورد نظر ببرید
2. **پایین آوردن**: ربات را به ارتفاع مناسب پایین بیاورید
3. **رها کردن**: دکمه **O** را بزنید تا گیره باز شود
4. **دور شدن**: ربات را از شیء دور کنید

## تنظیمات
- **رزولوشن**: 6.0 درجه در هر مرحله (ثابت)
- **خروج**: کلید **Esc**

## نکات مهم
- همیشه قبل از استفاده گیره را کالیبره کنید (کلید C)
- موقعیت گیره در رابط نمایش داده می‌شود
- وضعیت گیره (باز/بسته/در حال کالیبره) نمایش داده می‌شود
- برای کنترل دقیق‌تر از انتخاب مفصل + چرخ ماوس استفاده کنید
